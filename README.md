# CB
_A QOL module for hypixel skyblock dungeons_ 

## Features
- Cosmetic blocks for dungeons and kuudra (legit)
- FME for dungeons and kuudra (cheat)

## Installation 
Download fabric-api for 26.1.2 or 26.2

### Steps 
1. Download the lastest version of the mod
2. Move the jar file into your mod folder `minecraft\mods`
3. Restart Minecraft 

## Usage
_To open the GUI, the command is /cb!_ --- Enjoy!
